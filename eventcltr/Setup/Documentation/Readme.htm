About Likewise Enterprise

Likewise Enterprise is an enterprise software solution that improves management and interoperability of Windows®, Linux®, UNIX®, and Mac systems to considerably reduce IT costs and enhance security. Using familiar Windows-based tools, system administrators can easily and seamlessly integrate non-Windows systems with Microsoft® Active Directory™.
